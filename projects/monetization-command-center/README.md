# Monetization Command Center

Internal revenue-ops dashboard for the EM Fast-Profit Lab.

Open:

```text
projects/monetization-command-center/src/index.html
```

Use it to choose the weekly focus offer, calculate sprint revenue, generate outreach copy, export a pricing matrix, and follow a paid-delivery checklist.

## Operating Use

1. Choose one focus offer.
2. Set conservative outreach assumptions.
3. Generate the revenue plan.
4. Copy the script and personalize it for one prospect.
5. Deliver with the linked customer-facing project tool.
6. Track whether the offer hits the gate: two warm replies per 20 personalized messages, one paid pilot, and delivery under 60 minutes.
