# VibeCode Security Smoke Test Opportunity Analysis

AI-assisted development lets founders ship quickly, but many launches skip basic auth, secret, payment, upload, and LLM abuse checks.

## Why It Can Sell Quickly

- The buyer has launch urgency and clear fear of public embarrassment.
- The deliverable is narrow enough to finish quickly.
- Findings naturally create a retest and monthly release-gate service.

## MVP Scope

- Manual finding CSV
- Severity and launch gate scoring
- Remediation owner and retest plan
- Client-facing security report

## Expansion

Add automated route checks, secret scanning, dependency checks, Playwright probes, and GitHub PR release gates.
