Chargeback Evidence Kit is a single-purpose static MVP for merchants who need one clean dispute response packet. It converts evidence rows into a checklist, counterargument, customer outreach note, and owner brief.

## Buyer

## Outreach

I built a small tool that turns your existing rows into a finished Chargeback Evidence Kit deliverable. Send 5-10 sample rows and I will return a sample output before we talk about a monthly plan.

## Delivery Workflow

1. Collect sample rows.
2. Run the static MVP.
3. Review output for business rules and edge cases.
4. Export CSV/text and deliver the customer-ready copy.
5. Ask whether the same job recurs weekly or monthly.

## Guarantee

If the first pass does not produce a usable deliverable, revise the rules once or refund the setup fee.

## Upsell Trigger

Offer monthly support only when the customer has recurring rows, repeated policy changes, or multiple operators who need consistent copy.
